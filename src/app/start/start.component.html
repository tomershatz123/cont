This project was written as a home assignment for a company I was interviewed in. It is written in Angular 7

The user should upload 2 xlsx files: One of vessels and the other of telemetries. The application should read the files and find matching between event and vessel. A match is when an event row and vessel row both have the same longitude and latitude values, and Time and Received values should be at most 15 minutes difference. Upload the files from ./src/data directory.
